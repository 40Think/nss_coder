# Global Project Tags

## Purpose

Use special tags to indicate the role of each component in the global project.

## Tag Format

```python
# @TAG:GLOBAL-PROJECT:stage-1-mind-refactoring
# @TAG:DIGITAL-TWIN:decision-making-module
# @TAG:ASI-COMPONENT:strategic-planning-pattern
```

## Tag Categories

### Stage Tags
```
@TAG:GLOBAL-PROJECT:stage-1-mind-refactoring
@TAG:GLOBAL-PROJECT:stage-2-digital-twin
@TAG:GLOBAL-PROJECT:stage-3-asi-component
```

### Function Tags
```

@TAG:DIGITAL-TWIN:decision-making-module

```
